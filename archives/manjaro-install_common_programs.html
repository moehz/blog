# Google 输入法

  1. 安装 fcitx-googlepinyin , fcitx-im , fcitx-configtool

安装过程中只需要不断 回车 就行
2. 配置 .xprofile
在当前用户的家目录中创建 .xprofile

  1. logout 后重新登陆
    桌面右下角出现 键盘图标 ， 右键 点击 configure ，点击跳出窗户左下角的 + ，取消勾选 Only Show Current Language , 找到 Google Pinyin 并添加

  2. 按 ctrl+space 即可切换为谷歌拼音了

# ZY-Player

基于 electron 的跨平台桌面端视频资源播放器，简洁无广告，免费高颜值。支持很多网站，资源很全面。

# balenaEtcher

U 盘启动盘制作工具。

# kdiff3

文本对比工具。可以集成到 Dolphin 文件管理器中。

# Stacer

系统监视和优化工具。可以用它查看 cpu、内存、磁盘、网络 使用情况，管理进程和服务，管理开机启动项，管理软件包等等。

# Bleachbit

垃圾清理工具，自带商店可下载安装

# psensor

硬件温度监控工具，能监控 CPU、内存、硬盘、显卡等硬件，并在温度超过指定值时发出警报，自带商店可下载

# xdman

下载工具，安装对应的浏览器插件后浏览器就可以调用 xdm 进行下载

# simplescreenrecorder

录屏工具，自带商店可下载

# peek

gif 录制工具，自带商店可下载

# genymotion

安卓模拟器，基于 virtualbox。可以被 Android Studio 调用，自带商店可下载

# virtualbox

开源、高性能、轻量级的虚拟机软件，自带商店可下载
