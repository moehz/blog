从 Python 2.4 起，Python 标准库 string 引入了 Template 也可以用来格式化字符串。
所以说，与前三种方式的一个显著区别就是： Template 并属于 python 语言的核心语法特征，使用方式如下：

另外一个不同的地方是这个模板字符串不支持类似 str.format 那样的进制转换，需要我们自己处理

使用模板字符串 Template 的最佳的时机就是当你的程序需要处理由用户提供的输入内容时。模板字符串是最保险的选择，因为可以降低复杂性。
